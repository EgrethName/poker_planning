# poker-fe

## Tooling setup

Install [nvm](https://github.com/nvm-sh/nvm)
```
curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.36.0/install.sh | bash
```

Source nvm
```
source ~/.nvm/nvm.sh
```

Install node and npm
```
nvm install 12.13.1
npm install -g npm
```

npm install
